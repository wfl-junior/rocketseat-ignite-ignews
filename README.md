# Rocketseat Ignite ig.news

### [Clique aqui para visitar site em produção](https://wfl-junior-ignews.vercel.app)

Site para visualização de posts e artigos com autenticação pelo GitHub e paywall para visualização completa, permitindo apenas preview caso não tenha inscrição.

## Tech Stack

- [NextJS](https://nextjs.org)
- [TypeScript](https://www.typescriptlang.org)
- [NextAuth](https://next-auth.js.org)
- [Prismic CMS](https://prismic.io)
- [FaunaDB](https://fauna.com)
- [Stripe](https://stripe.com)
- [SASS](https://sass-lang.com)
- [React Testing Library](https://testing-library.com)
- [GitHub OAuth](https://developer.github.com/v3/oauth)
